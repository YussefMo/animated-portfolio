# Product

## Register

brand

## Users

Tech recruiters, senior engineering leads, and technical hiring managers evaluating Youssef Mohammed Ali as a Frontend Engineer candidate. They visit with intent — vetting credibility, scanning shipped work, and forming a hire/no-hire signal quickly.

In 5 seconds they should know: Frontend Engineer, React / Next.js / React Native, ships production-grade work.

## Product Purpose

A personal portfolio that converts scrutiny into conviction. It presents Youssef's identity, technical range, and shipped projects with the same precision he applies to production code. Success is a viewer who moves from "who is this?" to "I want to talk to this person" without friction.

## Brand Personality

**Precision, Depth, Restraint.** The site feels like expert engineering — nothing decorative, nothing accidental, nothing that a template would do. Confidence is earned through execution, not declared through flash.

- **Precision**: every interaction is deliberate; spacing, timing, and typography are tuned
- **Depth**: layered surfaces (WebGL, glass panels, staggered reveals) reward attention without demanding it
- **Restraint**: no over-animation, no gradient-everything, no scroll-jacking; the work speaks louder than the chrome

## References

- **Linear** — dark-mode command center; opinionated minimalism that feels fast and engineered
- **Vercel** — developer-first brand; clean typography, confident whitespace, product-forward
- **Figma** — technical but human; dark interface with purposeful color accents

## Anti-references

- Template/theme portfolios — same hero layout, same card grid, same "About / Work / Contact" monotony
- Flashy agency portfolios — over-animated, gradient-heavy, scroll-jacking, style over substance
- Junior dev portfolios — emoji clutter, hobby projects as experience, skills as progress bars, weak typography

## Design Principles

1. **Show the work, not the scaffolding.** The portfolio demonstrates engineering, not portfolio-trope assembly. Every component earns its place.
2. **Depth rewards attention.** Background layers, staggered reveals, and subtle interactivity reward exploration without demanding it.
3. **Confidence through restraint.** No bounce, no elastic, no gradient text. Motion is deliberate and serves content.
4. **Code is the brand.** The site's architecture (component separation, data layer, animation strategy) mirrors the professionalism claimed in the bio.
5. **First impression in 5 seconds.** Role, stack, and production credibility must survive a quick scan above the fold.

## Accessibility & Inclusion

- WCAG AA target for color contrast and interactive elements
- `prefers-reduced-motion: reduce` disables animations (already implemented in globals.css)
- Keyboard-navigable navigation (anchor links, visible focus indicators expected)
- Body text minimum 16px (Inter), code tags 12px minimum (JetBrains Mono)
